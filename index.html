About Us

SHHS Coding Club is dedicated to promoting a passion for programming and assisting other students in their coding endeavors. We aim to learn various programming languages, code independent and collaborative projects, and create a supportive environment for coding. We are currently focusing on the field of Web Development where we try to make personal portfolio websites on our own using HTML and CSS.
